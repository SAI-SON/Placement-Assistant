# **App Name**: ResumePilot

## Core Features:

- Resume Analysis & Scoring: Analyze uploaded resumes using the Gemini API to extract key sections, score the resume based on predefined criteria, and provide an overall quality assessment.
- Job Skill Matching: Compare parsed resume data against a database of company-specific skill requirements to determine the candidate's suitability for specific roles.  Show the percentage of skill match for various companies.
- Personalized Learning Suggestions: When a resume score is below 70%, suggest relevant YouTube videos. Use the Gemini API as a tool to understand the areas for improvement and tailor video recommendations.
- Resume Style Recommendations: Recommend resume styles based on experience level (e.g., fresher, experienced, designer), and integrate with Canva API or Google Drive templates.
- AI Feedback Chat: Enable users to chat with a "Resume Coach" powered by the Gemini API. The chatbot should give detailed feedback and provide example phrases based on user queries.
- Candidate Dashboard: Provide candidates with a dashboard displaying their resume score, feedback history, and recommended videos.

## Style Guidelines:

- Primary color: Strong purple (#9D4EDD). It communicates a sense of both creativity (relevant to design-focused users) and reliability (relevant to job seekers).

- Accent color: Pink (#EEA2D8), used sparingly to draw attention to key interactive elements.
- Headline font: 'Belleza' (sans-serif) for headlines, and 'Alegreya' (serif) for body text.
- Use clean, professional icons to represent different resume sections and features.
- Employ a clean, well-organized layout that prioritizes readability and ease of navigation.
- Incorporate subtle animations to provide feedback and enhance the user experience (e.g., progress bars, loading animations).
